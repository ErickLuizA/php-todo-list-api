## :rocket: Getting Started

## 1. Download the repository

```shell
 $ git clone https://github.com/ErickLuizA/php_todo_list.git
  
  or 

 $ gh repo clone ErickLuizA/php_todo_list
```

## 2. Installation

```shell
# Go into the repository
$ cd php_todo_list

# Install dependencies
$ composer install
```

## 3. Execution

```shell
$ composer dev
```
